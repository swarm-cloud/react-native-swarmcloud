## Installation

### Installing (React Native >= 0.60.0)

Install `react-native-swarmcloud`:

```shell script
yarn add react-native-swarmcloud
```

or

```shell script
npm i --save react-native-swarmcloud
```

Go to your **ios** folder and run:

```shell script
pod install
```
